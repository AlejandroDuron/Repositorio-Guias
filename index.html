Guías de Desarrollo

01

Guía 1: Fundamentos de HTML y CSS

En esta guía se identifica la estructura básica de un documento web. Se aplicar reglas de estilo CSS a los elementos HTML usando estilos embebidos y hojas de estilo externas. Se utilizan los selectores CSS básicos y las propiedades CSS para la utilización de fuentes y textos.

Ejercicio 1 Ejercicio 2 Ejercicio 3 Mi Interés
02

Guía 2: CSS Avanzado y Modelo de Cajas

Esta guía se centra en utilizar y combinar selectores CSS avanzados. Conocer la utilidad de las pseudo-clases y pseudo-elementos CSS. Crear páginas web utilizando el Modelo de Cajas, identificando las respectivas partes (margin, border, padding y content).

Ejercicio 1 Ejercicio 2 Biografía
03

Guía 3: Posicionamiento y Layouts

En este trabajo se utilizan los diferentes tipos de posicionamiento CSS y se diferencian apropiadamente. Se utilizan las propiedades CSS relacionadas con la visualización de elementos. Se demuestra habilidad en maquetar sitios web con diseños fijos y fluidos.

Oscares Menú Horizontal Planes Index Galería Análisis
04

Guía 4: Formularios y Diseño Responsivo

En esta guía se evidencia habilidad para crear y estilizar formularios utilizando la diversidad de controles de HTML5. Utilizar correctamente la meta-etiqueta viewport. Definición correcta las media queries para aplicar reglas de estilo en función del ancho del viewport del navegador. La utilización de los conocimientos adquiridos durante el transcurso de la asignatura para la construcción de páginas web responsivas.

Media Querrys Ejercicio anterior con Mediaquerrys
05

Guía 5: Introducción a JavaScript

En esta guia se adquiere dominio de los elementos sintácticos básicos del lenguaje JavaScript. Se es capaz de generar código HTML desde secuencias de comando de JavaScript. Habilidad para poder crear páginas web utilizando métodos básicos de interacción con el usuario. Se Utilizan correctamente estructuras selectivas y repetitivas en JavaScript.

Pagina principal Generar numero Aleatorio Operaciones basicas Generar tablas
06

Guía 6: Funciones y eventos en JS

En esta guia se adquiere dominio en la construcción de funciones con JavaScript, Se hace uso de parámetros o argumentos en las funciones que realiza. Se demuestra habilidad en el manejo de eventos usando funciones como controladores de eventos asociados a elementos del documento web.

Parte I (Estructuras de control y arreglos) Ejemplo de utilización de Funciones Parte II (Funciones) Ejemplo de Registro usando expresiones regulares
